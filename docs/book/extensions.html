Extensions

Extensions

Stuffer is a framework that handles uploads and downloads, the point is that you can, and should, extend it to suit your needs.

Core Extensions

Stuffer comes with some core extensions. They are basically features that you can opt-in in your stuffer-config.json file:

...
"extensions": [
    "prc-gzip",
    "mod-jimp",
    ...
],

Read more about the Stuffer configuration

Then you need to refer to each extension's documentation page to learn how to use it and how to configure it.
